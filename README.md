## Cómo iniciar el sistema

Para poner en marcha la aplicación, debes seguir este orden:

1. **Iniciar el Servidor:** Ejecuta la clase principal del servidor para que comience a escuchar conexiones entrantes -> Main.java
2. **Iniciar el/los Clientes:** Una vez iniciado el servidor, arranca las instancias de ClienteGrafico.java. Para poder registraste ya que no hay ningun usuario guardado con anterioridad deberás poner el nombre que desees y pulsar el boton de Registrarte, de tal manera que te mostrara tu contraseña si el usuario NO existe ya.


## JavaDoc

El javadoc esta ya de por si escrito en los comentarios de las funciones, pero si quisiera verse en formato web .html deberas ir a la carpeta correspondiente y abrir el archivo index.html. La web del javadoc fue generada automaticamente por el propio IDE usado, por lo que es posible que algunos de los comentarios del codigo no aparezca en el html
